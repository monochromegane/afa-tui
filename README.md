# AFA-TUI
=======

[![Actions Status](https://github.com/monochromegane/afa-tui/workflows/tagpr/badge.svg?branch=main)][actions]

AFA-TUI is a rich Text User Interface for [AFA](https://github.com/monochromegane.com/afa).

## Installation

```sh

```
